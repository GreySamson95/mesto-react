# Место
## Проектная работа
Проект сделал для демострации скилла студента, полученные в процессе обучения. В проекте используются навыки работы с JavaScript. 

При создании проекта учавствовали инструменты:
* JavaScript;
* Flexbox;
* Grid Layout;
* Validation;
* Api;
* React;
